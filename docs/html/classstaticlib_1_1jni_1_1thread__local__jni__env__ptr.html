Detailed Description

Smart pointer that wraps JNIEnv pointer and implements thread-local logic for it.

When this object is first created in the given thread - that thread is attached to JVM. All other instances of this object created in the same thread will point to the same JNIEnv. Thread is detached from JVM when the first-created object is destroyed.
